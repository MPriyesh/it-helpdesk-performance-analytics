# IT Helpdesk Performance Analytics

## Problem Statement
An IT services company is struggling to track helpdesk performance, 
identify SLA breaches, and understand workforce attrition patterns. 
This project analyzes 5,000+ support tickets, 500 employee records, 
and 20 department profiles to uncover operational bottlenecks, 
predict SLA breaches, and provide actionable insights through an 
interactive Power BI dashboard.

## Tools Used
| Tool | Purpose |
|------|---------|
| Excel | Data cleaning, pivot tables, KPI summary |
| SQL (MySQL) | Data loading, joins, business queries |
| Python | EDA, visualization, ML model |
| Power BI | Interactive 4-page dashboard |

## Dataset
| File | Rows | Description |
|------|------|-------------|
| departments.csv | 20 | Department info and budgets |
| employees.csv | 500 | Employee profiles and attrition |
| tickets.csv | 5,000 | IT support ticket records |

## Key Business Questions
1. Which department raises the most Critical tickets?
2. What % of tickets breach SLA by priority level?
3. Which employees resolve tickets fastest?
4. Is there a link between overtime and attrition?
5. What is the monthly trend of tickets raised vs resolved?
6. Can we predict SLA breach before resolution? (ML model)
